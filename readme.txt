Show installment plans box on WooCommerce product pages with admin settings and auto-inject under a chosen selector.

== Description ==
- Settings page under WooCommerce → اقساط
- Shortcode: [wc_installment_prices]
- Auto-insert the box under a target selector/class
- Monthly rate with APR or simple markup
- Down payment (percent or fixed)
- Template override via theme: create `rs-installments/box.php`

== Installation ==
1. Upload `rs-installments.zip` via WordPress Plugins → Add New → Upload.
2. Activate the plugin.
3. Configure under WooCommerce → اقساط.
